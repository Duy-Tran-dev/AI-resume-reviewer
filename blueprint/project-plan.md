# Project Plan

# Project Name

AI Resume Reviewer for Software Engineering Students

---

# Project Vision

AI Resume Reviewer is an AI-powered web application that helps university students improve their resumes specifically for software engineering internships and entry-level software engineering positions.

Instead of providing generic resume advice, the application evaluates resumes from the perspective of a technical recruiter and Applicant Tracking Systems (ATS), then generates actionable feedback and an improved version of the resume.

The application should help students understand **why** their resume is weak and **how** to improve it.

---

# Problem Statement

Many university students struggle to create resumes that effectively showcase their technical skills and projects.

Most students:

- don't know what recruiters expect
- don't know how ATS systems evaluate resumes
- write weak project descriptions
- fail to quantify achievements
- receive little or no professional feedback before applying

As a result, many qualified students are rejected before reaching the interview stage.

---

# Target Users

Primary Users

- Computer Science students
- Software Engineering students
- Information Technology students
- University students applying for software engineering internships
- University students applying for entry-level software engineering jobs

---

# Goals

The application should help students:

- improve ATS compatibility
- write stronger project descriptions
- improve technical communication
- identify resume weaknesses
- rewrite weak bullet points
- generate a more professional resume

The application should deliver results within 15 seconds.

---

# MVP Features

## Resume Upload

Users can upload a PDF resume.

---

## AI Resume Analysis

The AI should analyze:

- ATS compatibility
- Resume structure
- Formatting
- Grammar
- Technical skills section
- Projects section
- Work experience
- Bullet point quality
- Action verbs
- Quantifiable achievements
- Readability
- Professionalism

---

## Feedback Report

The AI should provide:

- Overall score
- Strengths
- Weaknesses
- Missing sections
- High-priority improvements
- Detailed explanations

Every recommendation should explain *why* it matters.

---

## Resume Regeneration

The AI generates an improved version of the resume while preserving truthful information.

The AI must never invent projects, experience, skills, or achievements.

---

# Non-Goals (Out of Scope)

The MVP will NOT include:

- User accounts
- Google Login
- GitHub Login
- Resume history
- Dashboard
- Team collaboration
- Subscription plans
- Payments
- Mobile application

These features may be implemented after the MVP.

---

# Success Criteria

The MVP is successful if a user can:

1. Upload a PDF resume.
2. Receive AI analysis within 15 seconds.
3. Understand why improvements are recommended.
4. Generate an improved resume.
5. Download the improved resume.
